## 1) What are browsers?

- Goal of a website: allow users to hit specific servers, get content files  
  (HTML / CSS / JavaScript), and render them on the screen.

- A browser is a software application that:
  - Sends requests to servers
  - Receives HTML, CSS, and JavaScript
  - Interprets and renders them on the screen

---

## 2) How do browsers render websites?

In simple steps:

1. Browser requests the website from a server  
2. Server sends HTML  
3. Browser parses HTML - builds the DOM  
4. Browser parses CSS - builds styles  
5. Browser combines structure + styles - layout  
6. Browser paints pixels on the screen  
7. JavaScript runs and adds interactivity

---

## 3) What purpose do HTML, CSS, and JavaScript serve?

### HTML → Structure
- What content exists
- Headings, text, buttons, images

### CSS → Appearance
- Colors, layout, spacing, animations

### JavaScript → Behavior
- Clicks
- Logic
- Data fetching
- Dynamic updates
